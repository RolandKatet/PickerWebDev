Thank you for installing Picker!

To start using Picker, just find a Slack channel, /invite @picker, and type /pick this will open the pick dialog for your to start making picks.

If you want to avoid using the dialog and type our your pick requests you may do so with leveraging the /pick and adding the option you would like to use after. For instance, if you would like to pick someone to go get coffee for the team you could type /pick coffee. This would then select someone from that channel to be in charge of caffeinating the team.

Get excited because there are quite a few ways in which you can use Picker!

To learn more about what you can do with Picker type /pick help
